# Apex One — Step 1

This is the first part of the full architectural walkthrough project.

## Included in Step 1

- React + Vite project
- React Three Fiber 3D scene
- Modern exterior tower
- Glass facade
- Floor selection
- Mobile rotation and zoom
- Landscape and reflecting pool
- Vercel-ready setup

## Run locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Upload this complete project to a new GitHub repository.
2. Go to Vercel.
3. Choose **Add New → Project**.
4. Import the GitHub repository.
5. Leave the detected framework as **Vite**.
6. Click **Deploy**.

Vercel automatically runs:

```bash
npm run build
```

The next step will add the smooth floor-entry transition and first-person interior.
